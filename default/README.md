# Math Operations

This repository contains basic math operations and their corresponding tests.

## Functions

- `add(a, b)`: Adds two numbers.
- `subtract(a, b)`: Subtracts two numbers.

## Workflow

The repository is integrated with CI/CD workflows for automated testing using `pytest`.

## Usage

1. Clone the repository.
2. Install dependencies from `requirements.txt`.
3. Run tests using `pytest`.

## CI/CD

The repository includes a GitHub Actions workflow for automated testing on every push or pull request to the `main` branch.
